# CIST Android App (Compose)

보건복지부 CIST 간편 검사를 위한 안드로이드 앱 샘플입니다.

## 포함 기능
- 첫 화면: **검사하기**, **개발자 후원하기** 버튼
- 세로(포트레이트) 고정
- 모든 화면 하단 Google 광고 배너(테스트 광고 ID)
- CIST 15문항 검사 + 문항 번호 네비게이션(응답 시 연한 초록)
- 전체 응답 완료 시 결과 확인 및 위험군 안내
- Play Billing 기반 후원 버튼 (₩1,000 / ₩5,000 / ₩10,000 / ₩50,000)

## 주의
- 현재 광고 ID는 Google 샘플 테스트 ID입니다.
- 인앱 결제 사용 시 Play Console에서 productId를 동일하게 등록해야 합니다.
- 문항/컷오프는 간편 버전이므로 실제 배포 전 최신 공식 기준으로 검증이 필요합니다.
